# Bun + Vite

### Стек:
##### Vite | React | Tailwindcss | TailwindUI

***

Зависимости:

```
bun install
```

## Начало

Запустить:
```
bun start
```

Режим разработчика:
```
bun dev
```

Собрать сайт:
```
bun run build
```
